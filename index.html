Architecture

Como o DiDi funciona por dentro

1

Mensagem chega via WhatsApp

Meta Cloud API via webhook n8n encaminha para o servidor FastAPI via Tailscale VPN.

2

Claude Agent SDK processa

Sessão streaming persistente. System prompt carrega tools essenciais, skills ativadas sob demanda via prompt caching.

3

Tools e MCPs executam

Google Calendar, Gmail, banco de dados de pagamentos, gerenciador de memória, sistema de viagens — via tools Python e servidores MCP.

4

Resposta formatada para WhatsApp

Formatada para leitura rápida no celular. Suporta imagens, áudios, documentos, localizações e mensagens interativas.
