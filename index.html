🏗️ Arquitetura

Como o DiDi funciona por dentro

1

Mensagem chega via WhatsApp

A mensagem é recebida pela Meta Cloud API via webhook n8n, que encaminha para o servidor FastAPI rodando na porta 7254 via Tailscale VPN.

2

Claude Agent SDK processa

O servidor mantém uma sessão streaming persistente com o Claude. O system prompt carrega tools essenciais e skills são ativadas sob demanda conforme o contexto da mensagem — otimizando prompt caching.

3

Tools e MCPs executam

O agente acessa Google Calendar, Gmail, banco de dados de pagamentos, gerenciador de memória, sistema de viagens e mais — tudo via tools Python e servidores MCP integrados.

4

Resposta formatada para WhatsApp

A resposta é formatada com emojis, negrito e listas — otimizada para leitura rápida no celular. Suporta envio de imagens, áudios, documentos, localizações e mensagens interativas.
